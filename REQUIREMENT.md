# KBO 데이터 분석 모델

### 프로그램 개발 사유
- 평소 한국 프로야구에 관심있어하는 개발자 BlackTurtle은 상황별 어떤 타자와 투수가 좋은지 궁금해하였다.
- 하지만 이런 상황에 대해서 사소한 지식만 있던 BlackTurtle은 어떠한 형식으로 개발해야 할지 몰랐다.
- UI 화면에서 소속팀, 상황별을 설정하면 어떤 선수가 제일 좋은지, 제일 안좋은지 표현해줘야 한다.
- 또한 투수와 타자의 실시간 상황에 대한 성적 비교도 보여지면 좋다.
- 결과에 대한 예측값이 사용자에게 출력되어야 하며, 결과 예측 성공 및 실패에 대한 값도 나와야한다.
